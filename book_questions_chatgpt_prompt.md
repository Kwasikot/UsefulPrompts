# Question formulation prompt
```
Formulate a comprehensive list of questions about the book (at least 150). The questions should be organized by chapter, with a block of questions for each chapter preceded by the chapter title. Cover all topics within each chapter with these questions. Conclude with a section of general questions covering the entire book.
```

# Markdown creation with Table of content prompt and the comments table
```
Create markdown document with the name A with Table of content with links on local headings of document A.  
Every heading is a question that you are extracted earlier from the book.


You need to

1.  Create a title with Chapter name
2.  answer every question of Chapter 2,
3.  Write the answer in a document A with heading #question_1 for example
4.  Create table of content for entire document with links on local headings of document A

example of document:

# Table of Contents

1.  [Introduction](#introduction)
2.  [Detailed Project Plan](#detailed-project-plan)
3.  [Contact Information](#contact-info)

## Introduction

This is the beginning of the document.

## Detailed Project Plan

This section contains the bulk of the information. Because it is a long section, linking to it from the top saves scrolling time.

## Contact Info

You can reach us at email@example.com.

Note that links in the table of content is not lead to chatgpt website...
```
